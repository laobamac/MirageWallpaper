「通用」分区集中了启动、更新、外观、音频、壁纸库目录和创意工坊相关的设置。

## 启动

- **开机时自动启动 Mirage**：登录后自动运行 Mirage 并恢复壁纸。

## 软件更新

Mirage 使用 Sparkle 处理更新：

- **自动检查并下载更新**：开启后在后台检查并下载更新。关闭后仍可用菜单栏的「检查更新…」手动检查。
- **接收测试版更新**：在正式更新之外，额外检查最新的测试版（`main` 分支滚动构建）。

更新真实性由内置的 Ed25519 公钥校验。两个架构各自使用独立的 appcast。

## 音频

- **全局音量**：所有壁纸音量的总控制（0–100%）。最终音量是单个壁纸音量与全局音量的叠加。
- **全局静音**：一键静音所有壁纸。

单个壁纸的音量与速度在[播放控制](/MirageWallpaper/wallpapers/playback/)中调整。

## 壁纸库目录

这里列出 Mirage 扫描壁纸的所有来源，每条显示标题、说明和实际路径，并可：

- **在访达中显示**：打开对应目录。
- **选择目录…**：为「Steam 创意工坊目录」或「导入壁纸目录」指定自定义位置。
- **恢复默认**：把自定义目录还原为默认位置。

「Mirage 下载目录」会标注为「当前下载位置」。各目录的完整说明见[数据目录](/MirageWallpaper/advanced/data-directories/)。

此外还有：

- **自动刷新壁纸库**：开启后 Mirage 自动重扫目录，并在壁纸库工具栏显示刷新按钮。

## 创意工坊 / Steam API

### Steam API 线路（仅中国大陆）

在中国大陆地区，设置中会出现线路选择：

- **Steam 官方 Web API**（`api.steampowered.com`）
- **SteamCF 镜像**

切换到镜像时会弹出警告：该镜像仅允许中国大陆用户访问，并非 Steam 官方服务，不保证安全性与可用性，且**只代理浏览 API，不能加速 SteamCMD 登录或壁纸下载**。

### Steam Web API Key

填入你自己的 32 位十六进制 API Key。未设置或格式无效时，会显示提醒并回退到共享的内置 Key。设置有效 Key 后会显示「已设置专属 API Key」。Key 仅用于本机请求创意工坊浏览数据，请勿分享。详见 [Steam Web API Key](/MirageWallpaper/workshop/api-key/)。

## macOS 与高级

- **调整菜单栏色调**：让菜单栏在动态壁纸下观感更协调。
- **详细日志（供调试）**：开启更详细的日志，便于排查问题。
- **重置所有设置**：把全部设置恢复默认（红色按钮，请谨慎）。
