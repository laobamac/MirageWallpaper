从主窗口顶部的**设置**按钮，或[菜单栏](/wallpapers/menubar/)的「设置」（`,`）打开设置窗口。窗口顶部有五个分区标签。

![Mirage 设置窗口的关于页面](/images/docs/settings-about.webp)

*真实的设置窗口；顶部标签可切换性能、通用、插件、屏保与关于。*

## 五个分区

| 分区 | 内容 |
| --- | --- |
| 性能 | 播放规则（省电策略）与渲染质量。见[性能与播放策略](/settings/performance/) |
| 通用 | 启动、更新、语言、外观、音频、壁纸库目录、创意工坊 API。见[通用与音频设置](/settings/general/) |
| 插件 | 扩展与插件相关。见[插件](/settings/plugins/) |
| 屏保 | 安装并配置动态屏保。见[动态屏保](/screensaver/overview/) |
| 关于 | 版本信息、致谢与项目链接 |

## 保存与取消

设置底部有「取消」和「好」两个按钮：

- **好**：保存改动并关闭窗口。
- **取消**：放弃改动并关闭窗口。

当存在尚未保存的改动时，底部会显示「已修改」提示。部分设置（如音量、帧率）在调整时会实时预览，但仍需点击「好」才会持久化保存。

## 重置所有设置

「通用 → 高级」中提供「重置」按钮，可把所有设置恢复为默认值。这会影响播放规则、质量、语言、外观、目录、API Key 等全部配置，请谨慎使用。
