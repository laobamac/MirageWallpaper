title: 性能与播放策略
description: 配置省电播放规则和场景渲染质量，在观感与功耗之间取得平衡。
---

「性能」分区分为两部分：**播放规则**（在特定场景自动调整播放）和**渲染质量**。

## 播放规则

Mirage 会持续监测系统状态，在下列情况下自动改变壁纸播放行为。每一项都可以单独选择动作。

| 触发条件 | 可选动作 |
| --- | --- |
| 其他应用获得焦点时 | 保持运行 / 静音 / 暂停 |
| 其他应用全屏时 | 保持运行 / 静音 / 暂停 / 停止（释放内存） |
| 其他应用播放音频时 | 保持运行 / 静音 / 暂停 |
| 显示器睡眠时 | 保持运行 / 暂停 / 停止（释放内存） |
| 笔记本使用电池时 | 保持运行 / 暂停 / 停止（释放内存） |

动作的含义：

- **保持运行**：不改变播放。
- **静音**：继续播放但静音。
- **暂停**：暂停画面（`speed` 归零）。
- **停止（释放内存）**：完全停止渲染进程以释放内存与显存。

### 多个条件同时命中

当多个条件同时满足时，Mirage 会取**最强**的动作生效。强弱顺序为：保持运行 < 静音 < 暂停 < 停止。例如同时处于电池供电和其他应用全屏，且分别设为「暂停」和「停止」，则实际执行「停止」。

:::tip[省电建议]
笔记本用户可以把「使用电池时」设为暂停或停止，把「其他应用全屏时」设为暂停或停止，能明显降低功耗与发热。这些是全局策略，与单个壁纸的[播放参数](/MirageWallpaper/wallpapers/playback/)相互独立。
:::

## 渲染质量

质量选项主要作用于**场景壁纸**（SceneWallpaper）。

### 质量预设

一排预设按钮可一键套用整组质量参数：**低 / 中 / 高 / 极致**。它们会同时调整抗锯齿、后处理、纹理分辨率、反射和帧率。选完预设后仍可单独微调下面的选项。

### 抗锯齿

- 关闭
- MSAA ×2
- MSAA ×4
- MSAA ×8

抗锯齿在**切换壁纸后**生效。

### 帧率

帧率可在 10–120 之间调节，**实时生效**。

- 超过 30 会提示较高帧率增加耗电。
- 超过 60 会红色警告，过高帧率显著增加耗电与占用。

对多数动态壁纸而言，30 帧已足够流畅且更省电。

### 音频频谱

「启用音频频谱（网页壁纸）」控制是否向网页壁纸提供音频频谱数据，供其做音频可视化。

## 相关设置

- 全局音量与全局静音在[通用与音频设置](/MirageWallpaper/settings/general/)。
- 更高级的进程优先级、显存耗尽时暂停、崩溃后重启等在设置的高级选项中。
