"Settings → Plugins" shows Mirage's plugin extension points. It's divided into two blocks:

![The real Plugins page in Mirage Settings](/images/docs/settings-plugins.webp)

*The Plugins page in the current release.*

- **Built-in**: plugins that ship with Mirage.
- **Third-party**: plugins provided externally.

## Current status

In the current version, the Plugins page is a placeholder:

- The Built-in block shows "No built-in plugins".
- The Third-party block shows "None".

In other words, Mirage doesn't yet support plugin installation or third-party extensions. This page reserves space for a future plugin system.

If a later version introduces plugin capabilities, the related installation, enabling, and configuration instructions will be added here.
