Open the settings window from the **Settings** button at the top of the main window, or from "Settings" (`,`) in the [menu bar](/en/wallpapers/menubar/). There are five section tabs at the top of the window.

## The five sections

| Section | Contents |
| --- | --- |
| Performance | Playback rules (power-saving strategy) and render quality. See [Performance and Playback Strategy](/en/settings/performance/) |
| General | Startup, updates, language, appearance, audio, wallpaper library directories, Workshop API. See [General and audio settings](/en/settings/general/) |
| Plugins | Extensions and plugins. See [Plugins](/en/settings/plugins/) |
| Screen Saver | Install and configure the live screen saver. See [Live Screen Saver](/en/screensaver/overview/) |
| About | Version info, acknowledgments, and project links |

## Save and cancel

There are "Cancel" and "OK" buttons at the bottom of Settings:

- **OK**: saves your changes and closes the window.
- **Cancel**: discards your changes and closes the window.

When there are unsaved changes, a "Modified" indicator appears at the bottom. Some settings (like volume and frame rate) preview in real time as you adjust them, but still need "OK" to persist.

## Reset all settings

"General → Advanced" provides a "Reset" button that restores all settings to their defaults. This affects everything, including playback rules, quality, language, appearance, directories, and API key, so use it with care.
