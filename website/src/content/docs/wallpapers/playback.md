---
title: 播放控制与自定义属性
description: 调整音量、速度、填充模式，以及每个壁纸自带的自定义属性。
---

选中壁纸后，可以在预览侧栏实时调整它的播放参数。这些设置按壁纸单独保存，下次应用同一壁纸时会自动恢复。

## 音量

调整当前壁纸的播放音量。音量为 0 等同于静音。最终输出还会受[设置中的主音量](/MirageWallpaper/settings/general/)与全局静音影响。

## 速度

调整播放速度（`speed`）。速度为 0 表示暂停，恢复非零速度即继续播放。

## 填充模式

控制壁纸如何适配屏幕：

| 模式 | 行为 |
| --- | --- |
| 填充（cover） | 铺满屏幕，必要时裁切边缘 |
| 适应（contain） | 完整显示内容，可能留边 |
| 拉伸（stretch） | 拉伸至铺满，可能改变比例 |

## 自定义属性

许多 Wallpaper Engine 作品在 `project.json` 中定义了**用户属性**（`general.properties`），例如颜色、开关、滑块、下拉选项、文字内容等。Mirage 会读取这些属性并在侧栏生成对应控件，你的改动会作为 `propertyOverrides` 保存，并实时下发给渲染进程。

支持的属性类型取决于作品定义，常见的有布尔开关、滑块数值、颜色、下拉组合框和文本输入。属性的具体含义由作者决定，详见[project.json 结构](/MirageWallpaper/formats/project-json/)。

## 设置的保存与恢复

每个壁纸的运行时状态（音量、速度、填充模式、属性覆盖）会随壁纸单独持久化。切换壁纸再切回来，或重启 Mirage，之前的调整都会恢复。

## 与全局播放策略的关系

上面这些是**单个壁纸**的播放参数。此外还有一层**全局播放策略**，在全屏应用、其他应用播放音频、屏幕休眠或电池供电等情况下自动接管播放行为（继续 / 静音 / 暂停 / 停止）。这部分在[性能与播放策略](/MirageWallpaper/settings/performance/)中配置。
