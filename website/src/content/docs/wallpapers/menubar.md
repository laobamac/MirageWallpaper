title: 菜单栏控制
description: 不打开主窗口，直接从系统菜单栏控制壁纸播放。
---

Mirage 在系统菜单栏提供一个状态图标，点击即可快速控制当前壁纸，无需打开主窗口。

## 菜单项

菜单栏图标的下拉菜单包含以下项目：

| 菜单项 | 快捷键 | 作用 |
| --- | --- | --- |
| 打开 Mirage | `O` | 显示主窗口 |
| 导入壁纸… | `I` | 打开导入面板，从文件夹导入壁纸 |
| 设置 | `,` | 打开设置窗口 |
| 检查更新… | — | 通过 Sparkle 手动检查更新 |
| 项目主页 | — | 打开 GitHub 项目页 |
| 静音 / 取消静音 | `M` | 切换当前壁纸静音 |
| 暂停 / 继续 | `P` | 切换播放与暂停 |
| 覆盖到所有显示器 | — | 把当前壁纸应用到所有屏幕 |
| 停止壁纸 | — | 停止播放并清除当前壁纸 |
| 退出 Mirage | `Q` | 退出应用 |

## 动态切换的菜单项

**静音**和**暂停**是动态项：

- 静音后菜单项变为「取消静音」，图标随之变化；取消后恢复为「静音」。
- 暂停后菜单项变为「继续」；恢复播放后变回「暂停」。

这两项与预览侧栏的音量、速度控制联动：无论从哪里操作，菜单文字都会保持同步。

## 图标与菜单栏色调

Mirage 使用自带的菜单栏图标。设置中的「调整菜单栏色调」选项会影响菜单栏在动态壁纸下的观感，详见[通用与音频设置](/MirageWallpaper/settings/general/)。
