---
title: 浏览创意工坊
description: 在创意工坊标签中按排序、标签和关键字浏览壁纸，并查看物品详情。
---

完成[设置](/workshop/overview/)后，切换到「创意工坊」标签即可浏览 Wallpaper Engine 的海量创意工坊内容。

## 排序方式

顶部可以按以下方式排序浏览：

| 排序 | 说明 |
| --- | --- |
| 热门趋势 | 近期（约 7 天）热度上升的内容 |
| 最新发布 | 按发布时间从新到旧 |
| 订阅最多 | 累计订阅数最高 |
| 评分最高 | 综合评分最高 |

## 标签分类

标签栏提供丰富的分类，例如：

动漫、自然、抽象、风景、科幻、卡通、赛博朋克、奇幻、女孩、游戏、动物、音乐、车辆、科技、复古、城市、太空、暗黑、像素、极简、水下、放松、中世纪等。

选择标签即可只看该分类的内容，可与排序、搜索组合使用。

## 搜索

搜索栏支持按关键字查找创意工坊物品，结合标签能进一步缩小范围。

## 分页与加载

结果分页加载（每页约 30 项），滚动或翻页即可加载更多。预览缩略图会被缓存，重复浏览时更快。

## 物品详情

点击一个物品可以查看详情：标题、作者、预览、标签、评分、描述等元数据。确认满意后即可下载，见[下载与管理](/workshop/download/)。

## 预设

部分创意工坊物品是**预设**——它们本身不包含完整壁纸，而是依赖另一个底层作品并覆盖其属性。Mirage 会识别预设并在需要时提示先下载其依赖，详见[预设](/workshop/presets/)。

## 内容分级

创意工坊内容带有分级信息。请注意甄别，Mirage 的本地[筛选面板](/wallpapers/library/)也支持按「所有人 / 轻度裸露 / 成人」过滤已下载的内容。

:::tip
如果浏览时频繁遇到繁忙或超时，多半是共享的内置 API Key 被限流。填写你自己的 [Steam Web API Key](/workshop/api-key/) 可显著改善。
:::
